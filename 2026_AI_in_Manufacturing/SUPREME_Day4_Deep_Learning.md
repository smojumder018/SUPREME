# SUPREME 2026 — Day 4 Deep Learning Tutorial

This tutorial introduces deep learning for manufacturing applications using PyTorch.

## Tutorial Overview

The notebook contains three main parts:

1. **PyTorch Basics for Beginners**  
   Introduction to tensors, neural networks, loss functions, optimizers, and the PyTorch training loop.

2. **Feedforward Neural Network for LPBF Data**  
   A simple FFNN model is used on the LPBF dataset to demonstrate deep learning with tabular manufacturing data.

3. **CNN for Coil Visual Inspection**  
   A convolutional neural network is used to classify grayscale coil inspection images as OK or Scrap.
